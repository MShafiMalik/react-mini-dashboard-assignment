# React Mini Dashboard Assignment

Single React application that combines:

- Task 1: API-based listing page (search, sort, loading/error states)
- Task 2: Form + validation flow (inline errors and success behavior)

## Stack

- React + TypeScript
- Tailwind CSS
- shadcn/ui
- Light/Dark theme support
- Redux Toolkit
- Vite
